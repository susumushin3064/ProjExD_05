# タンクサバイバー

## 実行環境の必要条件
* python >= 3.10
* pygame >= 2.1

## ゲームの概要
戦車を方向キーで操作し、敵の戦車を打ち落とし点数を稼ぐゲームである。

## ゲームの実装
### 共通基本機能
* 背景画像と戦車の画像
### 担当追加機能
* タイム指定、スコアによる機能(担当:千代田)
* 戦車機能(担当:古藤)
* 敵戦車(担当:笠原)
* 爆弾(担当:榎本)
* 壁、反射(担当:沼川)
